Overview
=========================
Configs for bash, git, etc.

Install
=========================
Clone repository with configs and execute build.sh:

    git clone git://github.com/tarmolov/configs.git ~/.configs
    ~/.configs/./build.sh --name='YOUR NAME' --email=EMAIL
